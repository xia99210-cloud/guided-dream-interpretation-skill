# Guided Dream Interpretation

一个引导式梦境探索 Skill。

它借鉴心理学中的体验取向和人本主义姿态，帮助用户通过多轮追问整理梦境中的感受、个人意义和现实处境连接。它不是心理咨询、诊断或治疗，也不使用“梦见某物=某种固定含义”的符号词典。

## 它做什么

- 完整听取用户梦境。
- 围绕梦里最有力量的情绪、画面和转折进行追问。
- 尊重用户自己的联想、修正和拒绝。
- 基于用户确认过的线索生成叙事报告。
- 在用户明确同意后，把梦境原文和报告保存到本地梦境档案馆。

## 它不做什么

- 不做心理或医学诊断。
- 不替代心理咨询或危机干预。
- 不套用固定梦境符号词典。
- 不自动上传数据。
- 不读取账号、Cookie 或本地凭据。
- 不自动发布、评论或运营任何平台账号。

## 隐私与保存

梦境档案馆使用本地文件 `assets/dreams-data.js` 保存记录。只有当用户在报告完成后同意“收藏进梦境档案馆”时，才会写入梦境原文和整理报告。

`assets/dream-archive.html` 只读取同目录下的 `dreams-data.js`，不进行网络上传。页面会把梦境标题、原文、标签和报告按纯文本转义后显示，避免用户内容被浏览器当作 HTML 执行。

## 使用边界

这个 Skill 适合用于梦境反思、自我探索和个人记录。若用户表达明确的现实自伤、自杀或伤人风险，应停止梦境分析，优先确认现实安全状态并提供合适的紧急支持资源。
