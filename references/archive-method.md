# 存档方法

## 目的

将一次完成的梦境探索写入 `assets/dreams-data.js`，使其出现在梦境档案馆页面。

## 何时执行

报告（第 6 步）生成后，问一句"要不要收藏进梦境档案馆？"。用户同意时执行。用户未请求或拒绝则不写入，不追问。

用户同意后，静默提炼字段并写入，不展示 title、core 等内部术语。写完直接发链接。

## 目标文件

`assets/dreams-data.js`（与 `assets/dream-archive.html` 同目录）。

文件结构：`window.DREAMS = [ {记录1}, {记录2}, ... ];`

新记录**追加在数组最前面**（按日期从新到旧）。

## 字段说明

```javascript
{
  date: "YYYY-MM-DD",      // 对话发生的日期
  title: "...",            // 这个梦的短名，见下方"如何提炼 title"
  mood: "A",              // 情绪区域（A-G），见下方"如何判断 mood"
  core: "...",             // 核心句（题眼），见下方"如何提炼 core"
  tags: ["...", "..."],    // 描述标签，3 个，见下方"如何提炼 tags"
  dream: "...",            // 用户讲述梦境的原文，保留口语
  report: "..."            // 完整报告正文，段落用 \n\n 分隔，引用段落以 "> " 开头
}
```

## 如何提炼各字段

### title

从报告中选一个最能概括这个梦的短名（3-8 个字）。优先使用报告里已有的章节性标题（如"白衣之下的出口"）；若报告没有章节标题，则从梦境的核心意象或情绪转折凝练一个。不要直接用梦中物件的罗列（如"孕妇、车祸、虫子"），要有意义感。

### mood

基于 Russell 环形情绪模型（效价×唤醒度），判断这个梦的主导情绪基调，输出一个字母（A-G）。七区按网格排列：正效价三档（高→中→低）、负效价三档（高→中→低），G 是落在正中心的"没有明确情绪"。

```
            高唤醒        中唤醒         低唤醒
正效价     A 明亮激活    B 怅然渴望    C 温柔安宁
负效价     D 紧绷激烈    E 怅惘不安    F 沉郁低落
模糊                G 混沌中性
```

- **A 明亮激活**（正效价 + 高唤醒）：兴奋、狂喜、自由释放、冲破束缚
- **B 怅然渴望**（正效价 + 中唤醒）：怀念、渴望、未完成的温柔、感动
- **C 温柔安宁**（正效价 + 低唤醒）：温暖、归属、安心、被接住
- **D 紧绷激烈**（负效价 + 高唤醒）：恐惧、愤怒、追逐、坠落、失控
- **E 怅惘不安**（负效价 + 中唤醒）：失望、隐隐不安、焦虑、愧疚、若有所失
- **F 沉郁低落**（负效价 + 低唤醒）：失落、压抑、无力、困住、孤独
- **G 混沌中性**（不确定）：荒诞无逻辑、情绪模糊、碎片感

判断优先级：
1. 用户在对话中确认的**主导感受**——不是梦的画面内容，而是"做完这个梦/醒来时/对话中确认的那个情绪"。
2. 如果用户的感受是复合的，以**报告最终落点**的情绪基调为准。
3. 效价本身都说不清（荒诞、碎片、连正负都定不了）的，归 G。

### core

报告中最有力量的那一句话——"换个用户就不成立的那一句"。通常是：
- 报告里用 `> ` 引用格式标出的那句
- 或最终整合出的、只属于这次对话的核心理解

它必须具体到这个人、这个梦。不能是通用的格言或鸡汤。

### tags

3 个标签，用于描述这个梦"关于什么"。

提取标准：抓梦的**核心主题**——反复出现或情绪落点所在的处境、关系、需要（如"归属""被抛下""失控""自我证明"）。不写梦里物件的罗列（不写"公交车""虫子"）。

### dream

用户最初讲述梦境的原文，保留口语和细节。不修改、不润色、不补充。

### report

报告正文。注意：
- 段落之间用 `\n\n` 分隔（不是真正的换行，是字面字符串）
- 引用段落（通常是核心句的展示）以 `> ` 开头
- 如果报告开头有一个短标题行（已被提取为 title），从 report 中删掉它，避免重复
- 字符串中的双引号用 `\"` 转义，或在数据文件中用反斜杠转义

## 写入示例

```javascript
window.DREAMS = [
  {
    date:"2026-06-09",
    title:"宿舍、足球和蓝天",
    mood:"B",
    core:"我可以暂时离开，也仍然属于这里。",
    tags:["归属","自由","渴望"],
    dream:"我梦见我搬到了一个像学生宿舍一样的地方……",
    report:"我觉得现在已经找到这个梦真正属于你的那根线了。\n\n这个梦所呈现的，可能不只是\"想有一个安全的住处\"……\n\n> 我可以暂时离开，也仍然属于这里。\n\n……（完整报告见 golden-cases.md）"
  },
  // ... 之前的记录保持不动
];
```

## 注意事项

- 只追加，不修改已有记录
- 写入后不需要改动 HTML 文件
- 写入成功后，在最终回复里给用户 `assets/dream-archive.html` 的可点击入口链接，说明这是查看档案馆的页面；不要给 `assets/dreams-data.js`
- 不要在 report 里保留 markdown 加粗（`**...**`）——页面不解析 markdown，直接输出纯文本
